# Описание

Airesto.ru — это приложение для ресторанов, которое упрощает их работу и включает несколько ключевых продуктов: систему управления живой очередью и бронированиями, а также коммуникационный центр для автоматизации взаимодействия с клиентами и повышения эффективности.

Тестовое задание заключается в разработке страницы просмотра бронирований для ресторана.

Для реализации необходимо использовать **Vue.js** версии 3+. Выполнение бонусных задач поможет вам прокачать навыки, продемонстрировать свои умения и повысить шансы на успешное прохождение отбора

# FAQ


1. **Нужно ли делать мобильную версию?**
   * Нет, не нужно
2. **Можно ли использовать готовые библиотеки для отображения таблицы?**
   * Да, если найдете библиотеку, поддерживающую весь необходимый функционал. Однако реализация без библиотек оценивается выше
3. **Можно ли использовать готовые библиотеки для ui элементов (кнопки, инпуты)?**
   * Да, можно
4. **Нужно ли использовать TypeScript?**
   * Можно писать на TypeScript, JavaScript или с JSDoc
5. **Обязательно ли выполнять доп.задания?**
   * Нет, не обязательно, но доп. задания играют роль как отличающий фактор, среди тех, кто выполнит тестовое
6. **Какие критерии оценивания задания?**
   * Корректность реализации, качество написанного кода, выполнение дополнительных пунктов
7. **Что будет в случае успешного выполнения тестового задания?**
   * По вашему резюме и решению тестового задания мы отберем 5-6 кандидатов, кто справился лучше остальных, и назначим с ними личные встречи, для финального найма
8. **Работаете ли с гражданами не из РФ?**
   * Да, работаем
